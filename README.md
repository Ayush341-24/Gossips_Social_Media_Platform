# Gossips – Where Stories Spread

## About

**Gossips** is a real-time social media platform inspired by modern social networking applications. It allows users to connect with people, share their thoughts, photos, and videos, interact through likes and comments, and communicate through real-time chat.

This project is being built from scratch using the **MERN Stack** as a learning journey to understand how production-level web applications are designed and developed.
